This is the most noticeable thing I have learned, you're looking at it right now! This is the thing I can build with, can immediately see results with. As such it's one of the things I've enjoyed most. As with the other tutorials, I'm sort of smashing the lego bricks together rather than building a to-scale model of Paris.
So what do I know?

  • Tagging (good Lord, so much tagging)
  • How to generate a HTML skeleton (snaps for Andy for this lovely reference guide)
  • Text formatting (because I can, I've put this into a handy table below)
  • Adding images/hyperlinks and the like
  • RELATIVE PATHING (my notes here devolve into waxing poetic about how gr9 it is to have knowledgeable people on hand to point me in the right direction)
  • Inline styling, and all that malarky (see that other table below, am quite fond of tables)
  • I made a v plain multiple page website dooberry with actual navigation bits, that was pretty cool, all about the <a href="ur link here yo">

This probably seems quite a small achievement, but to me, it's astonishing. This is a thing I would have flailed (hard) at a couple of weeks ago.
The best thing about HTML is being able to view what you've picked up straight away. I've made this lil' thing with HTML (with a little CSS help from w3). And that's great fun. Tweaking around the template to make these individual pages has been made possible because of my understanding.

Here are some tasty tables of tags I can remember...

Block Elements:

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header label a section of the page
div div divide sections of the page
ol ordered list create a list of ordered things (1,2,3 etc.)
ul unordered list create an unordered list (bulleted points etc.)

Inline Elements:

Tag Name Purpose Example Result
em emphasised make emphasised text <em>techinal sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make a hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code displays pieces of markup or source code <code><em></code> em
span span groups and applies styles to inline elements <span style="font-size: 20px; font-weight: bold; color: #006699;">Once upon a time,</span> there was a young woman who wished to learn how to code... Once upon a time, there was a young woman who wished to learn how to code...
li list defines a list item <li>Vengeful whale</li>
<li>Salty sailors</li>
<li>The names "Queequeg" and "Starbuck"</li>
  • Vengeful whale
  • Salty sailors
  • The names "Queequeg" and "Starbuck"

    Document Tags:

    Tag Name Purpose
    html HTML defines the doc type as HTML
    head head contains metadata such as: title, style etc.
    body body contains all the contents of an HTML document, such as: text, hyperlinks, images, tables, lists etc.
    meta meta provides metadata about the HTML document (not displayed on the page)
